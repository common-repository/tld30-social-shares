== Description ==  

This a very simple plugin for WordPress. What it does is add easy social networking sharing for each of your blog posts and pages. Share by Email, Twitter, Facebook, Myspace, and Digg. Also features a Print This button for easy printing of your documents. Social Shares will be displayed where ever you enter the Php code. <a target="_blank" href="http://tld30.com/webmaster/tools/wordpress-social-shares-plugin/" title="Install Help">Install Help</a>
